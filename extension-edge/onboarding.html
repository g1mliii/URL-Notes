Welcome to URL Notes

Open Edge Shortcuts

Quick guide to get you started. You can revisit this page from the extension menu anytime.

Keyboard Shortcuts

  1. Alt + E — Open the URL Notes popup
  2. Alt + N — Open popup directly in New Note mode

You can change these in Chrome’s Shortcuts page.

Context Menu & Multi-Highlight

  1. Select text on any page
  2. Right‑click → “URL Notes: Add to new note” to create a clipped bullet with a text fragment link
  3. Or choose “Add to existing note” to append to your latest note for that site
  4. Use "Toggle Multi-Highlight Mode" to select multiple text snippets at once

Features & Tips

  • Rich Text Editor: Format text with bold, underline, strikethrough, colors, and lists
  • AI Rewrite: Use AI to rewrite notes in different styles (formal, casual, professional, etc.)
  • Version History: Premium users can view and restore previous versions of notes
  • Citation Generator: Automatically generate citations in APA, MLA, Chicago, Harvard, and IEEE formats
  • Export Options: Export to JSON, Markdown, Obsidian, Plain Text, Notion, or Google Docs format
  • Cloud Sync: Premium users can sync notes across devices with end-to-end encryption
  • Notes are grouped by site. Switch views in the popup (This Site / This Page / All Notes)
  • Use tags in the editor to organize and search faster

Free vs Premium

Free Features:

  • Unlimited local notes
  • Rich text editing
  • Context menu clipping
  • Multi-highlight mode
  • Basic export options
  • Last 5 versions per note

Premium Features:

  • Cloud sync with encryption
  • Unlimited version history
  • AI rewrite capabilities
  • Citation generator
  • Advanced export formats
  • Ad-free experience

Getting Started

  1. Try the keyboard shortcuts to open the popup quickly
  2. Select text on any webpage and use the context menu
  3. Experiment with the rich text editor and formatting options
  4. Organize notes with tags for easy searching
  5. Explore export options in Settings
  6. Consider upgrading to Premium for cloud sync and AI features
