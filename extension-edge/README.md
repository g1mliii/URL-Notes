# Microsoft Edge Extension

This directory contains the Microsoft Edge version of the Anchored browser extension.

## Quick Setup

1. Open Microsoft Edge
2. Go to `edge://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this directory

## Edge vs Chrome

Edge uses the same Chromium engine as Chrome, so this extension is identical to the Chrome version with no code changes required. just a few updates

## Testing

Run `test-edge-setup.js` in the browser console to validate the extension setup.

For packaging and distribution, see `PACKAGING.md`.
