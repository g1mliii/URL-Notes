Help & Getting Started

Welcome to Anchored! Here's everything you need to know to get started with taking notes on any website.

Getting Started

1. Install the Browser Extension

Download and install the Anchored browser extension from the Chrome Web Store to start taking notes on any website.

Get Extension

2. Create Your Account

Sign up for a free account to sync your notes across devices and access premium features.

3. Start Taking Notes

Use keyboard shortcuts (Alt+E to open popup, Alt+N for new note) or right-click context menu to create notes on any webpage.

Key Features

Smart Organization

Notes are automatically organized by website domain and specific pages for easy retrieval.

Rich Text Editor

Format your notes with bold, italic, colors, lists, and more using our intuitive editor.

Multi-Highlight Mode

Select multiple text snippets from a webpage and add them all to a single note at once.

Context Menu Integration

Right-click on selected text to quickly add it to a new note or append to an existing one.

Cloud Sync (Premium)

Sync your notes across all devices with end-to-end encryption for $2.50/month.

AI Features (Premium)

Rewrite notes, generate summaries, and get AI assistance with 500 tokens per month.

Keyboard Shortcuts

Alt + E Open the Anchored popup
Alt + N Open popup directly in New Note mode

You can customize these shortcuts in Chrome's Extensions → Keyboard shortcuts page.

Frequently Asked Questions

How secure are my notes?

All your notes are encrypted with AES-256 encryption before being stored. We use a zero-knowledge architecture, meaning we never have access to your unencrypted notes.

Can I export my notes?

Yes! Free users can export to plain text, while Premium users have access to all formats including JSON, Markdown, Obsidian, Notion, and Google Docs.

What happens if I cancel my Premium subscription?

You'll retain access to Premium features until your current billing period ends. After that, you'll be moved to the free plan but keep all your notes.

How do I change my password?

Use the "Forgot Password" link on the login page. Your notes will remain accessible without any re-encryption needed.
